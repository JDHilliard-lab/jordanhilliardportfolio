Synopsis

The Ford Motor Co. project involved developing a comprehensive environmental graphic design system for the Dearborn Campus. The visual language was derived from the deconstruction of the iconic Ford oval, creating a framework of "The Whole, The System, and The Parts" that flexes across architectural and digital touchpoints.

Process
The Relational Framework

The system was developed by deconstructing the Ford oval into quadrants. This allowed for a visual language that could be reassembled into various dimensional forms, maintaining brand heritage while embracing modern architectural integration.

Process
Dimensional Installation

Translating 2D graphics into 3D space, the project utilized physical builds to create immersive branded environments that interact with the architecture of the campus.
